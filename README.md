# Prethivirajan S Portfolio

Premium portfolio built with React, Vite, JavaScript, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy

### Vercel

```bash
npm install -g vercel
vercel
```

### Netlify

```bash
npm install -g netlify-cli
netlify deploy
```

The project includes `vercel.json` and `netlify.toml` so both platforms can build from `dist` correctly.
